1.조건문 switch
	다중선택문 : 하나의 변수값을 평가하여 , 각 값에 대해 개별적인 처리를 하는 문장.
	
	1) 구조
		switch(변수){
		case 값1 : 		// 값 1과 변수의 값이 동일한 경우 아래 실행문들 실행~
			실행문들....
			break;		// 위 실행문들을  실행하고 swich 강제종료!!
		case 값2 : 
			실행문들....
			break;
		case 값3 : 
			실행문들....
			break;
		.....
		default : 
			실행문들......
			break;		// 마지막 break은 생략가능
		}
	2) 특징과 주의점
		1. switch 문으로 평가 할 수 있는 변수는 주로 정수형, String
			(int, char, String)
		2. case 다음에는 반드시 값이 하나가 와야한다.
		3. case 0:..... case 1:...... 와 같이 반드시 값을 정렬할 필요는 없다.
		4. default 는 생략가능
		5. case 별로 {}로 묶지 않는다. 콜론 : 으로 영역 인식.
		6. 다음 case 가 실행되지 않도록 할 경우 이전 case 마지막에 
			강제 종료를 뜻하는 break; 키워드를 작성

2. 반복문 while 
	1) 조건식을 먼저 검사한 후, TRUE 일 경우 실행하는 반복문 (루프)
	2) 실행문 영역 {} 안에서 반복(루프)를 종료 할 수있게 만들어 줘야한다.
	
	3) 구조
		#1. 반복의 횟수를 알 때
		변수 = 초기값;
		while(조건식) {
			반복해서 실행할 명령문 작성....
			증감식;
		}
			
		#2. 무한반복 : 반복의 횟수를 모르고 (실행시 매번 외부조건에 따라 달라져야 할 때)
		while(true) {
			반복해서 실행할 명령문.....
			if(조건) { break; } 	// 종료시점이 반드시 필요
			while(true){
				break; 	// 가장 가까운 반복문을 종료
				continu;	// 현재 진행중인 반복만 파기하고 다시 반복
			}
		}

3. 반복문 do-while
	선 실행 후 비교 : 실행 먼저한 후 조건 검사하는 형태
		
	1) 구조
		do {
			// 반복해서 실행할 명령문

		}while(조건식);

4. 반복문 for
	지정된 수만큼 반복할때 사용
	1) 구조	        1	     2	    4
		for( 초기식 ; 조건식 ; 증감식 ){
			반복 실행할 명령들... 	3
		}

		1  -> 2 -> 3 -> 4 -> 2 -> 3 -> 4 -> .... -> 2 (거짓 -> 종료)

		초기식 : for문 처음시작할때 한번만 실행
			초기식 i 는 for문 안에서만 동작하기에 중복되어도 사용가능
		조건식 : 결과가 true/false 되도록
		증감식 : 1씩 증/감할때는 증감연산자 주로 사용

		#1. for문 안에서 만들어진 i 변수는 밖에서 사용불가
		#2.	밖에서도 사용하려면 밖에서 선언
		#3. 초기값 생략 : 밖에서 먼저 만들어놓은 변수라면
			for문안에 초기식 생략가능, 이때, 반드시 세미콜론은 작성
